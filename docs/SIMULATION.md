# Simulation & Virtual Environments

Documentation for virtual testing and social VR integration.

## Unity3D Integration
The core simulation environment for robotics testing.

- **MCP Server**: `unity3d-mcp`.
- **Communication**: OSC + WebSocket.

## Resonite (Metaverse)
Advanced virtual collaboration and machine building.

- **Connector**: `resonite-mcp`.
- **Logic**: Federated via ProtoFlux agents.

## VRChat
Social interaction and avatar-based testing.

- **OSC Port**: 9000 (VRChat standard).
- **Control**: via `vrchat-mcp`.
